microBean™ Reference

The microBean™ Reference project provides classes and interfaces assisting with implementing contextual references.

Status

This project is currently experimental, in a pre-alpha state, and unsuitable for production use.

Compatibility

Until further notice, this project's APIs are subject to frequent backwards-incompatible signature and behavior changes, regardless of project version and without notice.

Requirements

microBean™ Reference requires a Java runtime of version 21 or higher.

Installation

microBean™ Reference is available on Maven Central. Include microBean™ Reference as a Maven dependency:

<dependency>
  <groupId>org.microbean</groupId>
  <artifactId>microbean-reference</artifactId>
  <!-- Always check https://search.maven.org/artifact/org.microbean/microbean-reference for up-to-date available versions. -->
  <version>0.0.3</version>
</dependency>

Documentation

Full documentation is available at microbean.github.io/microbean-reference.
